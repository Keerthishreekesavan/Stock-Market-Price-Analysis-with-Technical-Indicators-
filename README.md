# Stock-Market-Price-Analysis-with-Technical-Indicators-
Analyzed and worked with 5 years of historical stock data covering 491 companies, applying Pandas and NumPy to explore trends and evaluate key technical indicators such as RSI, SMA, EMA, MACD, Bollinger Bands, Volatility, and Drawdown.Developed clear visualizations using charts and graphs to interpret market behavior and support data driven financial analysis.The system supports both single company and multi-company analysis modes, enabling interactive comparison of up to five companies simultaneously for trend evaluation and performance insights.
